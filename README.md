# CLARAINOVA02

**CLARA** — Consultora de Legislação e Apoio a Rotinas Administrativas

Assistente especializada em SEI-Rio, SDP, licitações e procedimentos administrativos.

## 🛠️ Stack

- **React** 18 + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (estilização)
- **shadcn/ui** + **Radix UI** (componentes)
- **Supabase** (backend)
- **TanStack React Query** (gerenciamento de estado server-side)
- **Motion (Framer Motion)** (animações)
- **Lenis** (smooth scroll)
- **Zod** (validação)

## 🚀 Como rodar localmente

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```

## 🧪 Testes

```bash
npm run test
```
